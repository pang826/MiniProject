## 1. UI 시스템
#### (1) UI 위에서 하는 행동이 게임 오브젝트와 겹치지 않도록 하기
- Using UnityEngine.EventSystems; 를 선언하고 EventSystem.current.IsPointerOverGameObject()를 사용하면 <br> bool 값으로 UI 위에서 실행한 행동인지 알 수 있음
  <br> UI 위에서 했을 경우 true 값 반환
